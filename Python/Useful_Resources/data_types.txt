Data Casting

    1. Data to string - str(dataType)
    2. Data to integer - int(dataType)
    3. Data to float - float(dataType)

Instance function:
    isinstance() is a Python built-in function used to determine 
    whether an object belongs to a specified class or data type.

    examples: 
    # Check if a variable is an integer
    x = 5
    print(isinstance(x, int))  # Output: True

    # Check if a variable is a string
    y = "Hello"
    print(isinstance(y, str))  # Output: True

    # Check if a variable is a list
    z = [1, 2, 3]
    print(isinstance(z, list))  # Output: True

    # Check if a variable is a float
    a = 3.14
    print(isinstance(a, float))  # Output: True

    # Check if a variable is a dictionary
    b = {"name": "Alice", "age": 30}
    print(isinstance(b, dict))  # Output: True
